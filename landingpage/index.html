About Us

Learn More About Us

Ag-Food merupakan suatu software atau perangkat lunak yang dibuat untuk melakukan transaksi dan memasarkan usaha UMKM makanan di Nganjuk.

Ag-Food terdapat dua jenis software yaitu berupa mobile dan website.

  • Aplikasi mobile digunakan untuk costomer atau pembeli.
  • Aplikasi website digunakan untuk penjual atau Owner.
